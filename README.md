# Movie Recommendation System

A machine learning based movie recommendation system that suggests movies based on similarity algorithms.

## Features
- Content-based movie recommendation
- Uses cosine similarity
- Movie metadata processing
- Web interface using Flask
- TMDB API integration

## Tech Stack
- Python
- Flask
- Pandas
- Scikit-Learn
- HTML / CSS / JavaScript
- TMDB API

## How it Works
1. Movie dataset is processed.
2. Feature vectors are generated.
3. Cosine similarity matrix is calculated.
4. User enters a movie name.
5. System returns similar movies.
